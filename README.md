# Python Django Online Store

## Описание

Этот проект представляет собой веб-приложение интернет-магазина на основе Django. 
Основная цель проекта — предоставить пользователям удобную платформу для покупки товаров онлайн.


Основные функции:


Просмотр и поиск товаров по категориям.

Добавление товаров в корзину и оформление заказа.

Регистрация и аутентификация пользователей.

Управление профилем пользователя и историей заказов.


## Установка

Чтобы установить проект, выполните следующие шаги:

Клонируйте репозиторий:

git clone https://github.com/choz163/Python_Django.git
Копировать

Перейдите в директорию проекта:
cd Python_Django
Копировать

Установите зависимости:
pip install -r requirements.txt
Копировать
Выполните миграции базы данных:
python manage.py migrate
Копировать

Запустите сервер разработки:
python manage.py runserver
Копировать
Теперь вы можете открыть приложение в браузере по адресу http://127.0.0.1:8000/.

## Использование

После установки и запуска сервера вы можете взаимодействовать с интернет-магазином через веб-интерфейс. 
Зарегистрируйтесь, чтобы начать добавлять товары в корзину и оформлять заказы. 
Вы также можете просматривать доступные товары и управлять своим профилем.
